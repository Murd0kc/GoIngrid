# GoIngrid

Plataforma modular para aprender inglés desde cero hasta C2.

El proyecto es independiente de English Coach y tendrá su propio repositorio, base de datos, variables de entorno, contenedores y despliegue.

## Estado

Proyecto en fase de planificación. La documentación maestra define el currículo, la metodología, la práctica activa, la personalización, la IA, la pronunciación y las métricas.
